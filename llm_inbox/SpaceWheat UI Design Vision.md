# SpaceWheat UI Design Vision

## The Three-Layer Architecture

### **Layer 1: Strategic Empire View (The Stellaris Layer)**
*Turn-based grand strategy - Managing compressed nodes and quantum tunnels*

#### **Core Visual Language**
- **Galactic Map**: Nodes as floating crystalline structures connected by quantum tunnels
- **Node Representations**: 
  - **Stable nodes**: Warm golden glow (predictable resource generation)
  - **Chaotic nodes**: Flashing, flickering chaos patterns (unpredictable, potentially dangerous)
  - **Compressed nodes**: Perfect geometric forms (stable, efficient resource extraction)
- **Quantum Tunnels**: Shimmering bridges connecting biome simulations
- **Imperial Presence**: Massive golden structures representing Imperial demand

#### **Interface Elements**
- **"Next Turn" Button**: Measures all quantum systems, extracts aggregate resources, advances global state
- **Node Information Cards**: **Single-click** reveals stats that persist when zooming to biome
- **Resource Dashboard**: Global wheat/crystal/energy totals with trend indicators
- **Captain's Fleet Status**: Available subsidiary captains and their current assignments
- **Icon Attention Allocation**: Pie chart showing where your Wife's consciousness is focused

#### **Strategic Actions**
- **Quantum Tunnel Construction**: Major action binding biome simulations together
- **Icon Influence Adjustment**: Adjust Wife's philosophical attention focus across biomes
- **Resource Allocation**: Distribute accumulated resources between compressed nodes
- **Captain Deployment**: Assign subsidiary captains to manage distant nodes

---

### **Layer 2: Biome Temporal Manipulation (The Quantum Proof Laboratory)**
*A-temporal quantum probability field orchestration - Crafting the perfect timeline*

#### **Core Visual Language**
- **Quantum Sprites**: Beautiful non-interactive background elements showing vector state
- **Emoji Lattice**: Interactive grid where all agent actions occur
- **Temporal Visualization**: Quantum proof patterns rippling through time
- **Three Quantum Access Levels**:
  - **Classical Layer (Sparks/Operators)**: Real axis operations only - clean, geometric
  - **Quantum Angular Layer (Druids)**: Phase space manipulation - mystical, rotating
  - **Full Quantum Layer (Icons)**: Complete complex transformations - transcendent, impossible

### **Layer 2: Biome Temporal Manipulation - Input System**
*Quantum proof laboratory with intuitive keyboard controls*

#### **Core Input Philosophy**
- **Letters = Actions**: WASD camera, QE common, RF ultimate abilities
- **Numbers = Targeting**: 1-9 for quantum lattice nodes (max 10 qubits per biome)
- **Arrows = Temporal**: Left/right timeline, up/down speed control
- **Spacebar = Measurement**: Collapse timeline into reality

### **Layer 2: Biome Temporal Manipulation - Input System**
*Quantum proof laboratory with intuitive keyboard controls*

#### **Core Input Philosophy**
- **Letters = Actions**: WASD camera, QE common, RF ultimate abilities
- **Numbers = Targeting**: 1-9 for quantum lattice nodes (max 10 qubits per biome)
- **Arrows = Temporal**: Left/right timeline, up/down speed control
- **Spacebar = Measurement**: Double-tap for preview then confirm

#### **Movement & Camera**
- **WASD**: Standard camera movement through quantum biome
- **Mouse**: Pan and zoom for detailed lattice inspection
- **Tab**: Toggle quantum overlay (classical real axis ↔ quantum complex plane)

#### **Agent Deployment System**
- **Q**: Common operators (wheat farmers, resource chains) - assign your most-used stack
- **E**: Spark deployment - instant energy conversion abilities
- **F**: Druid spellcasting - quantum proof casting from simple to legendary
- **R**: Icon/Spouse ultimate - reality-shaping consciousness abilities

#### **Targeting Integration**
- **Numbers First**: Press 1-9 to select lattice node → available actions highlight
- **Letters First**: Press Q/E/F/R → compatible target numbers light up
- **Double-tap Number**: Zoom into specific quantum node for detailed view
- **Double-tap Letter**: Zoom to Heart layer focused on that agent type for detailed stats

#### **Mass Deployment**
- **Shift+Numbers**: Select multiple nodes → `Shift+1,2,3 → Q` deploys operators to nodes 1, 2, and 3
- **Shift+Letters**: Multiple deployments to one location → `Shift+Q,Q,Q,Q,Q → 5` dumps several operators on node 5

#### **Timeline Control System**
- **Arrow Keys**: Temporal navigation with Fibonacci speed scaling
- **Left/Right**: Skip through timeline (rewind/fast-forward)
- **Up/Down**: Speed control: -13x, -8x, -5x, -3x, -2x, -1x, -0.5x, 0 (pause), 0.5x, 1x, 2x, 3x, 5x, 8x, 13x
- **Timeline Slider**: Visual scrub bar at top of screen with temporal traces
  - **Left edge**: Previous measurement point
  - **Right edge**: Current simulation progress
  - **Temporal Traces**: Bright lines where sparks activated, color tints where druids were active
  - **Click anywhere**: Jump to that temporal position

#### **Temporal Bookmarks**
- **Alt+1 through Alt+9**: Bookmark current timeline position
- **1 through 9**: Jump back to bookmarked temporal moment (when not used for targeting)

#### **Measurement System**
- **Spacebar (first tap)**: Pause and preview measurement outcome
- **Spacebar (second tap)**: Confirm measurement - collapse quantum timeline into reality
- **Can measure at any point**: Rewind to ideal moment and measure there
- **Emergency measurement**: System getting chaotic? Rewind and measure at stable point
- **Timeline reset**: After measurement, timeline resets with new starting conditions

#### **Quality of Life**
- **Ctrl+Z**: Temporal undo - revert to previous agent placement without timeline scrubbing

#### **Visual Feedback Systems**
- **Temporal Ghosting**: Faint echoes showing where agents will be most effective across timeline
- **Force-Directed Lattice**: Emoji QuBit lattice auto-arranges into intuitive collections and connections
- **Selection Recentering**: Force-directed graph recenters on selected node (touchscreen-friendly)
- **Measurement Optimization Display**: Below timeline, colored bars with emoji indicators show optimal measurement moments for each resource type
- **Quantum Access Level Indicators**: Visual cues showing which agents can affect which nodes
- **Measurement Glow**: Nodes glow warmer (golden) for good measurement outcomes, cooler (blue/purple) for chaotic states

#### **Agent Deployment Mechanics**
- **Sparks**: Instant imaginary→real energy conversion (efficiency scales with total energy)
- **Operators**: Continuous real axis resource chains (quantum-blind production)
- **Druids**: Quantum proof casting from simple rotations to legendary topo mergers
- **Icons**: Full quantum consciousness evolution (global transformation rules)

#### **Temporal Proof Crafting**
1. **Timeline Exploration**: Scrub through time to understand quantum evolution
2. **Agent Orchestration**: Place agents at strategic temporal coordinates
3. **Proof Complexity**: Watch simple rotations evolve into complex quantum spells
4. **Cascade Observation**: See how quantum proofs ripple through timeline
5. **Timeline Collapse**: Spacebar measurement when proof sequence is perfect

#### **Core Visual Language**

- **Capacity Indicators**: Visual representation of each Socialite's customization bandwidth

- **Socialite Selection**: Choose which Socialite to work with
- **Enhancement Workspace**: Drag-and-drop interface for character modification
- **Capacity Management**: Visual tracking of Socialite customization limits
- **Discovery Implementation**: Convert Ace findings into deployable enhancements

#### **Socialite Specializations**
- **Different Socialites**: Each focuses on different enhancement types
- **Capacity System**: Limited customization bandwidth per Socialite
- **Meta-Customization**: Socialites can enhance other Socialites
- **Discovery Bridge**: Convert Ace discoveries into usable character enhancements

#### **Enhancement Categories**
- **Captain**: Strategic capabilities and away team management
- **Ace**: Exploration parameters and discovery rates (customizable but never controllable)
- **Spark**: Energy conversion efficiency and target specifications
- **Druid**: Proof complexity access and quantum spell patterns
- **Operator**: Production chain types and conversion rates
- **Icon**: Limited modification (Captain's Wife only, others discovered not customized)

## UI Flow and Interactions

### **Universal Click Pattern: Select → Activate**
- **Single Click**: Select and reveal information
- **Second Click**: Activate/Deploy/Confirm action
- **Works across**: Mouse, touchpad, keyboard (number keys for selection)

### **Strategic → Biome Transition**
- **Away Team Selection**: Choose which agents to deploy to biome (limited hotkey slots)
- **Single-click Node**: Reveals persistent information panel
- **Second-click Node**: Zooms into biome with selected away team
- **Campaign Persistence**: Agents remain with player throughout entire campaign
- **Information Persistence**: Strategic node data remains visible during biome work

### **Biome Layer - Temporal Manipulation**
- **Timeline Slider**: Full control over simulation time - fast forward, slow forward, pause, slow backward, fast backward
- **Non-linear Editing**: Place agents at any point in timeline, watch effects cascade forward
- **Spacebar Measurement**: Hit spacebar when timeline configuration looks optimal
- **Temporal Experimentation**: "What if I had placed that Spark 15 minutes ago?" - just scroll back and try it
- **Action Placement**: Agents can be placed/removed at any temporal coordinate

### **Agent Hotkey Layout**
- **Strategic Preparation**: Assign agents to numbered hotkeys before entering biome
- **Biome Deployment**: Number keys instantly place assigned agents at selected lattice points
- **Limited Slots**: Strategic constraint - can only bring limited number of agents per mission
- **Persistent Team**: Same agents grow and develop throughout campaign

### **Agent Card Integration**
- **Lattice Selection**: Single-click lattice node reveals applicable agent cards
- **Card Deployment**: Second-click agent card deploys to selected lattice node
- **Deck Management**: Heart button reveals full agent deck for organization
- **Hotkey Support**: Number keys for quick agent selection and deployment

### **Measurement Synchronization**
- **Biome "Measure"**: Collapses probability fields, extracts resources
- **Strategic "Next Turn"**: Measures all biomes, aggregates resources globally
- **Unified Mechanic**: Same measurement concept across both layers
